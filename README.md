# swimmer-detected
# 🏊 AI Swimmer Detection & Speed Analysis

An AI-based computer vision system for detecting, tracking, and analyzing swimmers from swimming videos using YOLO and OpenCV.

---

# 📌 Project Overview

This project focuses on:

- 🏊 Swimmer Detection
- 🎯 Single Swimmer Tracking
- ⏱ Swimming Time Measurement
- 🚀 Swimming Speed Estimation
- 🎥 Video-based Sports Analysis

The system allows users to:

- Select a target swimmer
- Track only one swimmer
- Measure swimming time
- Calculate average swimming speed over a 50m Olympic pool
- Export analyzed videos automatically

---

# 🧠 Technologies Used

| Technology | Purpose |
|---|---|
| Python | Main programming language |
| YOLO | Swimmer detection |
